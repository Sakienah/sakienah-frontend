# sakienah-frontend

Frontend voor Sakienah — e-commerce webshop.

## Starten

```bash
npm install
cp .env.example .env.local   # vul de waarden in
npm run dev
```

## Omgevingsvariabelen

| Variabele           | Beschrijving            |
| ------------------- | ----------------------- |
| NEXT_PUBLIC_API_URL | URL naar de backend API |
